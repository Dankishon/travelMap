# Legacy Files (Flask v2.0)

Эта папка содержит файлы из предыдущей версии проекта (Flask + vanilla JS).

**ВНИМАНИЕ:** Эти файлы больше не используются. Актуальная версия находится в корне проекта.

## Содержимое

- `app.py` - старый Flask backend
- `index.html`, `script.js`, `style.css` - старый vanilla JS frontend  
- `requirements.txt` - старые зависимости Flask
- `docker-compose.yml` - старый Docker Compose для Flask
- `Dockerfile` - старый Dockerfile для Flask
- `init_data.py` - старый скрипт инициализации
- `data/` - старые JSON файлы данных
- `README.md`, `QUICKSTART.md` - старая документация
- `CHANGELOG.md`, `DOCKER.md`, `START_DOCKER.md` - исторические документы
