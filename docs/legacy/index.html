О проекте TravelMap Russia

TravelMap Russia — это простое веб-приложение для планирования туристических маршрутов по России.

Возможности:
  • Интерактивная карта России с интересными местами
  • Просмотр информации о достопримечательностях
  • Создание собственного маршрута
  • Сохранение маршрута на сервере
Технологии:
  • HTML, CSS, JavaScript
  • Bootstrap 5 — фреймворк для интерфейса
  • MapLibre GL JS — библиотека для интерактивных карт
  • OpenStreetMap — картографические данные
  • Flask — backend сервер
Проект создан для демонстрации возможностей веб-разработки.
